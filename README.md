# 🔮 ガチャ占い - 全占術統合システム

by クトちゃん 💫

## 📱 このアプリについて

スマホゲームのガチャ運を全占術で占うWebアプリです。

### 占術統合
- 西洋占星術
- 数秘術
- タロット占い
- 四柱推命
- 九星気学
- 血液型占い

### 主な機能

✅ **ゲームタイトル選択**
- プリセット：ウマ娘、ブルアカ、原神、NIKKE、など
- 「その他」で任意のゲームタイトルを手入力可能

✅ **詳細設定**
- 狙っているキャラ/武器名
- ガチャタイトル（コラボ名など）
- ガチャ期限（日付＋時刻まで設定可能）
- 個人情報（生年月日、星座、血液型、干支）

✅ **占い結果**
- 総合運気スコア（100点満点）
- 最適ガチャ日時を3つ提示
- 推奨ガチャ回数（単発・10連・合計）
- 全占術の詳細解説
- ガチャ前のラッキーアクション

✅ **特別機能**
- 🎨 幸運の女神画像表示（運気アップ）
- 📄 PDF出力機能（結果を保存）
- ⏰ 過去の時間帯を自動判定（「終了」表示）
- 🔄 時間経過で結果が自動更新

## 🚀 使い方

### オンラインで使う（GitHub Pages）

このリポジトリをGitHub Pagesで公開すると、以下のようなURLでアクセスできます：

```
https://[あなたのユーザー名].github.io/gacha-fortune/
```

### ローカルで使う

1. `index.html` をダウンロード
2. ブラウザで開く

## 📖 GitHub Pagesでの公開手順

### 1. このリポジトリをFork/Clone

```bash
git clone https://github.com/[あなたのユーザー名]/gacha-fortune.git
```

### 2. GitHub Pagesを有効化

1. リポジトリの「Settings」タブを開く
2. 左メニューの「Pages」をクリック
3. Source: 「Deploy from a branch」を選択
4. Branch: 「main」を選択
5. Folder: 「/(root)」を選択
6. 「Save」をクリック

### 3. 数分待つ

数分後、以下のURLでアクセスできるようになります：

```
https://[あなたのユーザー名].github.io/gacha-fortune/
```
